# OpenCoze

#### Description
OpenCoze，中文名称开源扣子。
借助开源手段，通过一站式数字化系统，实现企业管理各环节、各用户之间的紧密协作。
项目构成：供应链（采购、销售、库存），WMS模块、MES模块、供应商平台、财务模块
项目底座版本Odoo18.0。
